Building a multiroom chat application in Node JS, Express JS, Socket.io, MongoDb

Nkululeko Mokoena

2021-07-16

Github:https://github.com/NkululekoMo/Chatting-App

In this project, I used the Node.js, Socket.io, Express.js platform to build a real-time chat application that sends and shows messages to a recipient instantly without any page refresh. Also, it have some validations like if the username is taken then we will have to choose some other username. If two users are in the same room then only they can see the message of each other

Conclusion:

As you can see is pretty simple and it doesn’t require tons of code as I thought before building it.

The power of Socket.IO extends to more solutions that previously relied on back-end pages. Now, the inbound/outbound server response time can rely on the WebSocket protocol for more agile information exchange, enabling systems that track monitoring results or even setting real-time pricing for users.
